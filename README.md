# Simple Calculator 🧮

This is a beginner-friendly Python project that performs basic arithmetic operations.

## 🔹 Features:
- Addition
- Subtraction
- Multiplication
- Division (with zero-division check)

## 📌 How to Use:
1. Run the program
2. Choose the operation
3. Enter two numbers
4. The result will be displayed

## 🛠️ Built With:
- Python 3
